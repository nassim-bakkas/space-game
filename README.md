> A 2D space shooter game built with **Java 17** and **JavaFX 21**.

---

## 🎮 Gameplay

- Control a spaceship and shoot down enemies
- Dodge bombs and collect power-ups
- Survive waves of enemies and achieve the highest score
- Animated star field background and explosion effects

---

## 🛠️ Tech Stack

| | |
|---|---|
| Language | Java 17 |
| UI Framework | JavaFX 21 |
| Build Tool | Maven |

---

## 🚀 Getting Started

### Prerequisites
- Java 17+
- Maven

```bash
git clone https://github.com/nassim-bakkas/space-game.git
cd space-game
mvn clean javafx:run
```

---

## 📁 Project Structure

```
src/main/java/com/spacegame/
├── MainApp.java        # Entry point
├── GameView.java       # Main game loop & rendering
├── MenuView.java       # Main menu
├── GameOverView.java   # Game over screen
├── Player.java         # Player spaceship
├── Enemy.java          # Enemy logic
├── Projectile.java     # Bullets
├── Bomb.java           # Enemy bombs
├── PowerUp.java        # Power-up items
├── Explosion.java      # Explosion animations
└── StarField.java      # Animated background
```
